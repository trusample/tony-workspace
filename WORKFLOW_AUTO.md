# Workflow for CRM Migration and Enhancement

## Current Tasks
1. **Create or access new Google Sheet**
   - Title: "TruSample CRM Master"
   - Bind Apps Script to new sheet.

2. **Copy necessary data from legacy CRM**
   - Tabs: Accounts, Contacts, Quotes, Orders, Products

3. **Update existing scripts**
   - Ensure that all necessary functions and API calls work correctly with current data structures.

4. **Ensure correct permissions and scope for all operations**

5. **Deploy updated Apps Script project**
   - Confirm with Maykel for any additional requirements.
